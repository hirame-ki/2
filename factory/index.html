Features
교사의 하루를 더 가볍게
H-Factory는 교사의 일상 업무를 돕기 위해 설계된 위젯 앱입니다. 설치하고 바탕화면에 올려두기만 하면 됩니다.
Feature 01
🏫
NEIS 자동 연동
학교 급식 메뉴와 학사일정을 NEIS에서 자동으로 불러옵니다. API 키 없이도 기본 사용이 가능합니다.
Feature 02
⏱️
교시 타이머
현재 교시와 남은 시간을 실시간으로 표시합니다. 시간표와 연동되어 다음 수업도 미리 알려줍니다.
Feature 03
🎲
수업 도구 모음
무작위 추첨, 타이머, 사다리타기, 룰렛, QR코드, 뽀모도로까지. 수업 진행에 필요한 도구를 한곳에서.
Feature 04
🗂️
업무도구 모음
상담 일지, 진도 체크, 성적 분석, 수업자료 생성까지. 학급 담임 교사의 핵심 업무를 별도 창에서 깊이 있게 관리할 수 있습니다. 성적 분석은 1차 정기고사 일람표로 가상 등급 산출도 지원합니다.

How to Install
설치 방법은 단 4단계
Step 01
🔗
다운로드 페이지 열기
아래 다운로드 버튼을 클릭하면 GitHub Releases 페이지로 이동합니다. 로그인 없이 누구나 무료로 받을 수 있습니다.
Step 02
📦
.exe 파일 다운로드
페이지 하단 Assets 목록에서 H-Factory.Setup.x.x.x.exe 파일만 클릭하여 받으세요. 다른 파일(.blockmap, .yml 등)은 필요 없습니다.
Step 03
🛡️
Windows 보안 경고 허용
처음 실행 시 파란색 보안 경고 팝업이 나타날 수 있습니다. "추가 정보"를 클릭한 후 "실행"을 누르면 됩니다. 악성 소프트웨어가 아니니 안심하세요.
Step 04
🎉
바탕화면에서 시작!
설치가 완료되면 H-Factory가 바탕화면 위에 바로 실행됩니다. 이후 업데이트는 앱에서 자동으로 알려줍니다.
⚠️
파일이 "확인되지 않음 XXXXXX.crdownload"로 저장되나요?
Chrome/Edge 브라우저가 .exe 파일을 위험하다고 판단해 차단한 것입니다. 실제 악성 소프트웨어가 아니니 안심하세요. 아래 세 가지 방법 중 하나로 해결할 수 있습니다.

방법 1 — 파일 이름 바꾸기 (가장 빠름)
다운로드 폴더에서 .crdownload 파일을 우클릭 → 이름 바꾸기 → H-Factory.Setup.exe 로 변경 → 확장자 경고 팝업에서 예 클릭 → 실행

방법 2 — 브라우저 다운로드 목록에서 허용
Chrome 주소창에 chrome://downloads 입력 → 해당 파일 옆 ⋮ 메뉴 → 유지 또는 허용 클릭

How to Use
업무도구 사용법

업무도구 위젯(🗂️)의 도구 버튼을 클릭하면 각 도구가 별도 창으로 열립니다.

💬
상담 일지

학생별 상담 내용을 날짜별로 기록하고 조회하는 도구입니다.

기본 사용법
  1. 좌측 패널에서 반 선택 후 상담할 학생 클릭
  2. 우측 상담 기록 패널에서 + 상담 추가 버튼 클릭
  3. 날짜, 상담 유형(학업 / 생활 / 진로 / 기타), 상담 내용 입력 후 저장
  4. 저장된 상담 기록은 날짜 역순으로 타임라인 형식으로 표시됩니다
주요 기능
  • 상담 횟수 배지: 학생 목록에서 각 학생의 누적 상담 횟수 확인 가능
  • 이름 검색: 상단 검색창으로 학생 빠르게 찾기
  • 상담 유형 색상 코딩: 학업(파랑) / 생활(초록) / 진로(주황) / 기타(회색)
  • 엑셀 내보내기: 전체 학생 요약 시트 + 학생별 개별 시트로 내보내기 가능
💡상담 일지 데이터는 H-Factory 설정 파일에 자동 저장됩니다. 앱을 재실행해도 데이터가 유지됩니다.
📋
진도 체크

반별 단원·차시 진도를 기록하고 관리하는 도구입니다.

기본 사용법
  1. 상단에서 과목 탭 추가 (＋ 버튼)
  2. 상단 우측에서 반 버튼 추가
  3. 좌측 사이드바에서 단원 추가 버튼으로 단원 등록
  4. 우측 콘텐츠 영역에서 각 단원의 ＋ 차시 추가 버튼으로 차시 등록
  5. 각 차시별로 미완료 / 진행중 / 완료 상태 선택
주요 기능
  • 과목별, 반별로 진도 상태를 독립적으로 관리
  • 단원 카드에 진행률 표시 (프로그레스 바 + 퍼센트)
  • 하단 요약 바에서 전체 차시 완료/진행중/미완료 현황 확인
  • 데이터는 브라우저 로컬스토리지에 자동 저장
💡처음 실행 시 과목과 반이 비어 있습니다. 상단 ＋ 버튼으로 직접 추가해주세요.
📊
성적 분석

NEIS에서 내보낸 성적 파일을 분석하여 학생별 성적 변화 추이와 AI 상담 프롬프트를 생성하는 도구입니다. 교과학습발달상황과 전과목 성적 일람표(1차 정기고사) 두 가지 파일 형식을 모두 지원합니다.

엑셀 파일 다운로드 방법 (NEIS)
① 교과학습발달상황: 나이스 ▸ 학급담임 ▸ 학교생활기록부 ▸ 학생부 항목별 조회 ▸ 교과학습발달상황(전체학생)
② 전과목 성적 일람표: 나이스 ▸ 학급담임 ▸ 성적조회/통계 ▸ 학기말성적조회 ▸ 성적일람표 전과목 (산출기준: 1차 정기시험 / 받은점수기준)

위 경로에서 .xlsx 파일을 다운로드하여 사용합니다.

사용 방법
  1. 파일 종류 칩에서 교과학습발달상황 또는 전과목 성적 일람표 선택
  2. 업로드 존에 NEIS 성적 파일(.xlsx)을 드래그하거나 클릭하여 선택
  3. 드롭다운에서 분석할 학생 선택
  4. 원하는 과목 칩을 선택/해제하여 분석 범위 지정
  5. 학생의 진로 희망 직업/계열 입력 (선택)
  6. 📊 분석 & 프롬프트 생성 버튼 클릭
분석 결과
  • 학생 요약: 분석 과목 수, 평균 등급, 최고/최저 등급 요약 카드
  • 가상등급 분석 (일람표 전용): 원점수·석차 기반으로 가상 5등급(고1~2) · 9등급(고3) 자동 산출. 한 등급 올리기 위해 앞에 있는 학생이 몇 명인지 과목별로 표시
  • 아쉬운 과목 순위: 추월 인원이 적은 과목 순으로 정렬 → 어떤 과목에 집중하면 효율적으로 등급을 올릴 수 있는지 한눈에 파악, 상담 자료로 바로 활용
  • 성적 변화 추이 그래프: 계열별(전체/국어/수학/영어/사회/과학/기타) 탭으로 선 그래프 확인
  • AI 상담 프롬프트 자동 생성: Claude, ChatGPT, Gemini에 붙여넣어 성적 분석·진로 추천·상담 포인트를 받을 수 있습니다
  • 엑셀 다운로드: 분석 결과를 .xlsx 파일로 저장 가능
⚠️5등급제(고1~2 공통과목)와 9등급제(고3 선택과목)를 자동으로 감지하여 적용합니다.
📝
수업자료 생성 v1.5.0 신규

2022 개정교육과정 전체 성취기준(중학교 25과목 + 고등학교 151과목)이 내장된 AI 프롬프트 생성기입니다. 수업 주제와 성취기준을 선택하면 수업 자료 프롬프트를 자동으로 만들어 줍니다.

생성 가능한 자료
  • ① 수업 PPT 슬라이드 — HTML/CSS 웹 슬라이드 형식으로 Claude가 직접 생성
  • ② 교사 발문 — 밑줄·색깔 강조가 포함된 Word 문서
  • ③ 학생 활동지 — 바로 사용 가능한 Word 문서
  • 그 외: OX퀴즈·수행평가·토론·모둠활동지·롤플레이·카드뉴스·개념노트·웹게임 등
사용 방법
  1. ① 과목·내용·성취기준 탭에서 학교급·과목·학년·수업 주제 선택
  2. 성취기준 목록에서 해당 기준 클릭하여 선택 (선택 사항)
  3. ② 자료형태 탭에서 원하는 결과물 칩 선택
  4. ③ 세부설정 탭에서 교수법·난이도·어조·디자인 테마 설정
  5. ④ 생성 탭에서 ✨ 프롬프트 생성하기 클릭 → 자동 복사
  6. claude.ai에서 Ctrl+V로 붙여넣기 — 기존 학습지·PPT도 함께 업로드하면 더 높은 완성도
💡모든 프롬프트에 할루시네이션 방지 지침이 자동으로 포함됩니다. 생성 기록은 최대 20개까지 저장됩니다.
📊
간단 성적 산출

지필·수행 항목별 반영비율을 설정하고 점수를 입력하면 100점 환산 점수, 등수, 등급을 자동으로 산출합니다. 엑셀 양식 다운로드 후 점수를 입력해 업로드하는 방식도 지원합니다.

기본 사용법
  1. 항목 설정 탭에서 평가 항목(예: 지필1, 수행1)과 만점, 반영비율(%) 입력 — 합계가 100%가 되도록 설정
  2. 채점표 탭에서 반 선택 → 명렬표에서 학생 자동 불러오기
  3. 각 학생의 항목별 점수 직접 입력 또는 엑셀 업로드로 일괄 입력
  4. 상단 성적산출 버튼 클릭 → 환산점수·등수·등급 자동 계산
  5. 필요 시 엑셀 내보내기로 결과 저장
주요 기능
  • 반영비율 환산: 각 항목 점수 ÷ 만점 × 반영비율 합산 → 100점 기준 환산점수 자동 계산
  • 등급 선택: 9등급제 / 5등급제 그때그때 선택 가능 (고3↔나머지 학년 유연하게 사용)
  • 정렬: 번호순 / 등수순 / 점수순 토글
  • 엑셀 양식 다운로드: 반 학생 명단이 포함된 입력 양식 생성 → 작성 후 업로드로 점수 일괄 적용
  • 통계 표시: 성적산출 후 인원·평균·최고·최저 요약
💡항목 설정과 점수 데이터는 브라우저 로컬스토리지에 저장됩니다. 창을 닫고 다시 열어도 유지됩니다.
📅
학급 이벤트 플래너

학급 행사, 시험, 체험학습, 학생 생일 등을 등록하고 D-DAY 위젯에 자동 연동합니다. 당일 앱 실행 시 팝업 알림도 제공합니다.

기본 사용법
  1. 이벤트 추가 탭에서 이름·날짜·유형·관련 반 입력
  2. 디데이 위젯에 자동 등록 토글 켜고 등록 → D-DAY 위젯에 즉시 반영
  3. 생일 관리 탭에서 학생 이름·생일(MM-DD) 입력 → 올해/내년 기준 날짜 자동 계산 후 D-DAY 등록
  4. 이벤트 목록에서 미등록 이벤트는 📌 버튼으로 언제든 D-DAY에 추가 가능
주요 기능
  • D-DAY 자동 연동: 등록한 이벤트·생일이 바탕화면 D-DAY 위젯에 실시간 반영
  • 당일 팝업 알림: 앱 실행 시 오늘 해당하는 이벤트·생일을 중앙 팝업으로 알림 (하루 1회)
  • 유형별 필터: 생일·시험·체험학습·행사·기타로 분류 및 필터링
